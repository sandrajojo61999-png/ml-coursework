# ML Coursework
A collection of Jupyter notebooks covering key topics in machine learning, from classical methods to modern deep learning architectures. Developed as part of my Master's studies.

## Topics

| Notebook | Description |
|----------|-------------|
| **Flow Matching** | Conditional flow matching for MNIST inpainting — masking and reconstructing image halves |
| **GMM** | Gaussian Mixture Models and the Expectation-Maximization algorithm |
| **Generative Models** | Flow matching on CIFAR-10 with a U-Net-style velocity network and sinusoidal time embeddings |
| **Graph Neural Networks** | 2-layer GCN for node classification on the Cora citation dataset (~80% test accuracy) |
| **Hyperparameter Optimization** | Brute-force search over SVM (two moons) and Ridge regression (California Housing) |
| **Independent Component Analysis** | FastICA from scratch in NumPy with PCA whitening; blind source separation on synthetic signals |
| **LLMs** | Small Transformer word embedding model trained on a text corpus, visualized with t-SNE/PCA; bonus: ViT image embeddings on CIFAR-10 |
| **Optimizers** | Comparing SGD (various LRs ± momentum), Adam, AdamW, and RMSprop on a fully connected network classifying CIFAR-10 |
| **Support Vector Machines** | SVM from scratch with SGD and hinge loss; tested on make_blobs, make_moons, and the Wine dataset |
| **Trees** | Classification tree (Gini impurity), Random Forest, and Extra Trees — all from scratch in NumPy, evaluated on Iris |
| **Vision Model** | Transfer learning with pre-trained VGG-11 on CIFAR-10; small U-Net for image segmentation on the Oxford-IIIT Pet Dataset |
| **XGBoost** | XGBoost from scratch in NumPy — regression on California Housing and binary classification on Breast Cancer |
