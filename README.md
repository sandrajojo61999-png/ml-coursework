# ML Coursework

A collection of Jupyter notebooks covering key topics in machine learning, from classical methods to modern deep learning architectures. Developed as part of my Master's studies.

## Topics

| Notebook | Description |
|----------|-------------|
| **Flow Matching** | Continuous normalizing flows and flow-based generative modeling |
| **GMM** | Gaussian Mixture Models and the Expectation-Maximization algorithm |
| **Generative Models** | Overview of generative modeling approaches (VAEs, GANs, etc.) |
| **Graph Neural Networks** | Message passing, node classification, and graph-level tasks |
| **Hyperparameter Optimization** | Grid search, random search, Bayesian optimization strategies |
| **Independent Component Analysis** | Blind source separation and ICA fundamentals |
| **LLMs** | Large Language Models — architecture, training, and fine-tuning |
| **Optimizers** | Gradient descent variants (SGD, Adam, RMSProp, etc.) |
| **Support Vector Machines** | SVMs for classification with kernel methods |
| **Trees** | Decision trees, random forests, and ensemble methods |
| **Vision Model** | Convolutional architectures and vision transformer approaches |
| **XGBoost** | Extreme Gradient Boosting for tabular data |

## Tech Stack

- Python
- Jupyter Notebook
- NumPy, Pandas, Matplotlib
- Scikit-learn
- PyTorch
